# Advanced Login Form UI

This project is part of my Full Stack Web Development learning journey.

A modern login interface built using only HTML and CSS.

---

## Features

- Responsive layout
- Animated login card
- Floating input labels
- Password strength UI bar
- Remember me checkbox
- Forgot password link
- Social login buttons
- Divider with OR text
- Gradient background
- Button hover animation

---

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Animations
- Media Queries

---


---

## How to Run

Open `index.html` in your browser.
