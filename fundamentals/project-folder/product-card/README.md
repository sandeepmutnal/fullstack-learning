# Product Card UI

This is a simple HTML + CSS project created as part of my Full Stack learning journey.

## Project Description
A product card component that displays:
- Product image
- Product name
- Description
- Price
- Buy button with hover effect

This project helps practice:
- HTML structure
- CSS styling
- Box model
- Flexbox alignment
- Shadows and hover effects

## Technologies Used
- HTML5
- CSS3
